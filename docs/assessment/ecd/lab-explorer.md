## Game Overview

- **Mechanic:** Open-ended science experiment simulation with guided challenges. Students choose equipment, set variables, run experiments, record observations, draw conclusions.
- **Subject:** Science (Chemistry, Physics)
- **Duration:** 15-20 min per session
- **Primary Analytics Dimension:** D3 (Strategic Behavior — experimentation rate, process mining, self-correction)
