## Game Overview

- **Mechanic:** Tower defense where towers are built/upgraded by answering questions. Enemy types represent specific knowledge components.
- **Subject:** Mathematics (number sense, operations, fractions, geometry)
- **Duration:** 10-15 min per session
- **Primary Analytics Dimension:** D3 (Strategic Behavior — resource allocation, problem-solving paths)

## 1. Competency Model

| Competency | Description |
|-----------|-------------|

| **Resource Allocation** | Strategic investment in tower types reveals planning and prioritization skills. |
| **Productive Persistence** | Whether students change strategy after wave failures vs. repeat identical approaches. |
